# 🐼 PANDA — Ethical Hacking & Security Research

> Personal project exploring offensive security concepts, ethical hacking techniques, and system protection strategies.

---

## 📌 About

PANDA is my self-directed cybersecurity learning project where I document, experiment with, and build tools around ethical hacking concepts. The goal is simple: understand how systems get broken so I can understand how to protect them.

This repo reflects my curiosity-driven approach to learning — no courses, just digging in, breaking things (responsibly), and figuring out how it all works.

---

## 🔍 Areas of Exploration

- **Reconnaissance** — passive and active information gathering techniques
- **Network scanning** — understanding how port scanning and service enumeration works
- **Vulnerability concepts** — studying how common vulnerabilities (SQLi, XSS, etc.) are found and exploited
- **CTF challenges** — practicing with Capture The Flag exercises to sharpen skills
- **Scripting for security** — writing custom scripts to automate security tasks
